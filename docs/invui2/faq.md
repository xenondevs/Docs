# Frequently Asked Questions

??? question "What Minecraft versions does InvUI v2 support?"
    Starting with v2.0.0, InvUI no longer supports multiple Minecraft versions. Instead, new InvUI versions will always only target the latest Minecraft version. You can find a table mapping InvUI versions to Minecraft versions in the [GitHub readme](https://github.com/NichtStudioCode/InvUI/tree/ver/2.x?tab=readme-ov-file#version-compatibility).

??? question "Will InvUI v1 be updated to Minecraft 26.1?"
    No InvUI v1 is end-of-life and will no longer receive **any** updates.
    If you need multi-version or Spigot support, you will need to switch to a different library. A list of alternatives can be found [here](https://www.spigotmc.org/wiki/creating-a-gui-inventory/#libraries).

??? question "Where can I find the old InvUI v1 documentation?"
    You can find the old InvUI v1 documentation [here](https://docs.xenondevs.xyz/invui1/).
